# Chalcatongo_Mapa

Mapa interactivo de cambios de uso de suelo en Chalcatongo, Oaxaca. Comparativa 2000 - 2013, 2013 - 2024.
Información obtenida mediante un enfoque postclasificatorio. Las clasificaciones de 2000, 2013 y 2024 se obtuvieron mediante una clasificación supervisada sobre mosaicos anuales sin nubes de imágenes Landsat. Toda la cartografía fue generada por Fernando Jiménez.

El archivo de interactivemap_builder.R contiene el código de R con el que se construyó el mapa de Leaflet que se muestra en la página de Github pages (ver enlace en la esquina superior derecha).
